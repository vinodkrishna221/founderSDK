## Responsive Design Specifications

### Desktop (1200px+)
- **Grid**: 3 columns with 32px gaps
- **Card Width**: 320px fixed width
- **Typography**: Full scale (20px H3, 14px body)

### Tablet (768px - 1199px)
- **Grid**: 2 columns with 24px gaps
- **Card Width**: Flexible (45% each)
- **Typography**: Medium scale (18px H3, 14px body)

### Mobile (320px - 767px)
- **Grid**: Single column stack
- **Card Width**: Full width minus 32px padding
- **Typography**: Small scale (16px H3, 14px body)
- **Buttons**: Full-width CTAs

## Animation Specifications

### Page Load Animations
- **Filter Tabs**: Slide in from top (0.4s ease-out)
- **Learning Cards**: Staggered fade-in (0.1s, 0.2s, 0.3s delays)
- **Progress Bars**: Animated fill on card hover

### Hover Effects
- **Cards**: translateY(-4px) with shadow increase
- **Progress Bars**: Subtle glow effect
- **Icons**: scale(1.05) with color transition

### Progress Animations
- **Module Completion**: Celebration animation with confetti
- **Progress Bar Updates**: Smooth width transitions
- **Achievement Badges**: Bounce animation on unlock

## Accessibility Features

### Keyboard Navigation
- **Tab Order**: Logical sequence through filter tabs and cards
- **Focus Indicators**: Clear visual focus states
- **Enter/Space**: Activate cards and buttons

### Screen Reader Support
- **Card Labels**: Descriptive ARIA labels including progress
- **Progress Bars**: ARIA value attributes
- **Status Updates**: Live region announcements

### Visual Accessibility
- **High Contrast**: Alternative color schemes
- **Large Text**: Scalable typography
- **Color Independence**: Progress shown through text and bars
